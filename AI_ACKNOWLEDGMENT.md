# AI Acknowledgment

In accordance with the MathWorks Excellence in Innovation Challenge guidelines, this document outlines the use of AI tools in the development of the FOMC AI Analyzer project.

## 1. Code Generation & Architecture
- **Tool Used:** Google Gemini 2.5 Pro / Anthropic Claude / AI Coding Assistants
- **Application:** Used for scaffolding the FastAPI backend, Next.js frontend components, and configuring the ChromaDB vector store. AI was heavily used for iterating on the Server-Sent Events (SSE) streaming logic and the React UI components (like the Thinking Panel).

## 2. Text and Content Generation
- **Tool Used:** Large Language Models
- **Application:** Used to generate the prompt engineering templates for the RAG pipeline, draft portions of the project README, and formulate the evaluation questions.

## 3. MATLAB Scripts
- **Tool Used:** AI Assistants
- **Application:** Assisted in structuring the MATLAB analytics layer and applying the Text Analytics Toolbox and Statistics and Machine Learning Toolbox to the FOMC data.

**Declaration:** All AI-generated code, text, and architecture decisions were manually reviewed, tested, and modified by the project author to ensure correctness, security, and alignment with the MathWorks Challenge #258 requirements.
